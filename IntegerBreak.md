# 题目
给定一个正整数 n，将其拆分为至少两个正整数的和，并使这些整数的乘积最大化。 返回你可以获得的最大乘积。

例如，给定 n = 2，返回1（2 = 1 + 1）；给定 n = 10，返回36（10 = 3 + 3 + 4）。

注意：你可以假设 n 不小于2且不大于58。
# 解法
```ruby
class Solution {
public:
    int integerBreak(int n) {
    if(n == 2) return 1;
    if(n == 3) return 2;
    int ret = 1;
    while( n>4 )
    {
        ret *= 3;
        n -= 3;
    }
    return ret * n;
}
};
```
# 分析
当作数学题来处理，可证得分出的3越多越好（4可以由两个2代替，可由计算得2比3更好，1不能选择为因子）
